# Diagrams

This directory contains system architecture, data flow, and timing diagrams for the FPGA-EKG design.

## Contents
- **Architecture:** System-level block diagrams and FPGA hardware modules.
- **Data Flow:** Signal processing pipeline from ADC inputs to filters and EKG feature detection.
- **Timing:** Waveform/timing diagrams detailing clock domains and bus interfaces (e.g. SPI, I2C).
