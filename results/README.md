# Results

This directory tracks the validation, benchmarking, and performance analysis of the FPGA-EKG design.

## Target Metrics
- **Accuracy:** Performance against standard datasets (e.g. MIT-BIH Arrhythmia Database).
- **Resource Usage:** Logic element utilization, register counts, RAM blocks, DSP block counts.
- **Latency:** Execution time of various filters and detection algorithms.
- **Plots:** MATLAB/Python generated charts showing input vs. output signals.
